# Validate Implementation

Run validation checks on the implementation.

## Checks to Run

1. **TypeScript Compilation**
   ```bash
   bun run typecheck
   ```

2. **Linting** (if configured)
   ```bash
   bun run lint
   ```

3. **Tests**
   ```bash
   bun test
   ```

## Your Mission

1. Run each validation check
2. If there are errors, fix them
3. Re-run checks until all pass
4. Report the final status of all checks

Do not proceed until all checks pass.
